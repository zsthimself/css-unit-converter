What is Point (PT) and Viewport Height (VH)?

Point (PT) is an absolute unit used in typography, while Viewport Height (VH) is a percentage unit relative to the viewport height.

Usage Instructions

  1. Enter the point value
  2. Click "Convert" button or press Enter to get the result
  3. Use "Copy" button to copy the conversion result

Common Conversion Examples

  • 1pt = 0.0139vh (assuming viewport height is 1000px)
  • 12pt = 0.1667vh

  • 72pt = 0.75vh
