What is Point (PT) and REM?

Point (PT) is an absolute unit, commonly used in printing and typesetting. REM is a relative unit, relative to the font size of the root element.

Usage Instructions

  1. Enter the point value
  2. Click "Convert" button or press Enter key to get the result
  3. Use "Copy" button to copy the conversion result

Common Conversion Examples

  • 1pt = 0.75rem (when root font size is 16px)
  • 12pt = 9rem
  • 24pt = 18rem
  • 6pt = 4.5rem
