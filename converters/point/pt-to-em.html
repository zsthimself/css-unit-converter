What is Point (PT) and EM?

Point (PT) is an absolute unit, commonly used in printing and typesetting. EM is a relative unit, relative to the font size of the parent element.

Usage Guide

  1. Enter the point value
  2. Click "Convert" button or press Enter key to get the result
  3. Use "Copy" button to copy the conversion result

Common Conversion Examples

  • 1pt = 0.75em (when parent element font size is 16px)
  • 12pt = 9em
  • 24pt = 18em
  • 6pt = 4.5em
