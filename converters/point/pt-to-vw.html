What is a Point (PT) and Viewport Width (VW)?

A point (PT) is an absolute unit used in typography and design. Viewport width (VW) is a percentage unit relative to the viewport width.

Usage Instructions

  1. Enter the point value
  2. Click "Convert" button or press Enter to get the result
  3. Use "Copy" button to copy the conversion result

Common Conversion Examples

  • 1pt = 0.0139vw (assuming viewport width is 1000px)
  • 12pt = 0.1667vw
