What is a Point (PT) and a Pixel (PX)?

A point (PT) is an absolute unit used in printing and typesetting. A pixel (PX) is the basic unit of screen display.

Usage Instructions

  1. Enter the point value
  2. Click "Convert" button or press Enter to get the result
  3. Use "Copy" button to copy the conversion result

Common Conversion Examples

  • 1pt = 1.33px
  • 12pt = 1in
  • 28.35pt = 1cm
  • 2.835pt = 1mm
  • 12pt = 1pc
  • 12pt = 1em
  • 864pt = 1ft
