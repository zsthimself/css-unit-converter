What are Viewport Width (VW) and EM?

Viewport width (VW) is a percentage unit relative to the viewport width. EM is a unit relative to the font size of the parent element.

Usage Guide

  1. Enter the viewport width value
  2. Click "Convert" button or press Enter to get the result
  3. Use "Copy" button to copy the result

Common Conversion Examples

  • 1vw = 0.1em (assuming root element font size is 16px)
  • 2vw = 0.2em
  • 5vw = 0.5em
  • 10vw = 1em
