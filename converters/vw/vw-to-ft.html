What are Viewport Width (VW) and Feet (FT)?

Viewport width (VW) is a percentage unit relative to the viewport width. Feet (FT) is a commonly used unit of length.

Usage Guide

  1. Enter the viewport width value
  2. Click "Convert" button or press Enter to get the result
  3. Use "Copy" button to copy the result

Common Conversion Examples
