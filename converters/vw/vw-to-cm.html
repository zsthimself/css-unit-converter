What are Viewport Width (VW) and Centimeters (CM)?

Viewport width (VW) is a percentage unit relative to the viewport width. Centimeters (CM) is a commonly used unit of length.

Usage Guide

  1. Enter the viewport width value
  2. Click "Convert" button or press Enter key to get the result
  3. Use "Copy" button to copy the conversion result

Common Conversion Examples

  • 1vw = 0.01cm (assuming viewport width is 1000px)
  • 2vw = 0.02cm
  • 5vw = 0.05cm
  • 10vw = 0.1cm
