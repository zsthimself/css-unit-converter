REM to Millimeter Conversion Instructions

REM is a unit relative to the root element font size. By entering the REM value and the root element font size, you can calculate the corresponding millimeter value.

Common Conversion Values (Based on 16px Root Font):

  • 1rem = 25.4mm
  • 2rem = 50.8mm
  • 6rem = 152.4mm

Usage Instructions:

  1. Set the root element font size (default 16px)
  2. Adjust the screen DPI value (default 96)
  3. Enter the REM value
  4. Get the corresponding millimeter value
