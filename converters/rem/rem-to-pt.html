REM to Point Conversion Information

REM is a unit relative to the font size of the root element. By entering a REM value, you can calculate the corresponding point value.

Common Conversion Values (Based on 16px Root Font):

  • 1rem = 12pt
  • 2rem = 24pt
  • 0.5rem = 6pt

Usage Instructions:

  1. Set the root element font size (default 16px)
  2. Enter the REM value
  3. Get the corresponding point value

Usage Tips:

  • Calculated based on the root font size
