REM to Centimeter Conversion Instructions

REM is a unit relative to the font size of the root element. By entering a REM value and the font size of the root element, you can calculate the corresponding centimeter value.

Common Conversion Values (Based on 16px Root Font):

  • 1rem = 2.54cm
  • 2rem = 5.08cm
  • 6rem = 15.24cm

Usage Instructions:

  1. Set the font size of the root element (default 16px)
  2. Adjust the screen DPI value (default 96)
  3. Enter the REM value
  4. Get the corresponding centimeter value

Usage Tips:

  • Suitable for print design
  • Calculate based on the font size of the root element
  • Accurate to 3 decimal places
