CSS单位转换器

支持各种CSS单位之间的相互转换，让前端开发更轻松

PIXEL (PX)

像素单位转换器

px → inches 96px = 1in px → rem 16px = 1rem px → em 16px = 1em px → cm 37.8px = 1cm px → mm 3.78px = 1mm px → pt 1.33px = 1pt px → pc 16px = 1pc px → vw 基于视窗宽度 px → vh 基于视窗高度 px → ft 1152px = 1ft px → % 基于父元素

INCH (IN)

英寸单位转换器

in → pixels 1in = 96px in → cm 1in = 2.54cm in → mm 1in = 25.4mm in → pt 1in = 72pt in → pc 1in = 6pc in → ft 12in = 1ft in → rem 1in = 6rem in → em 1in = 6em in → vw 基于视窗宽度 in → vh 基于视窗高度 in → % 基于父元素

REM

根元素相对单位转换器

rem → pixels 1rem = 16px rem → em 1rem = 1em rem → % 基于父元素 rem → inches 1rem = 0.167in rem → cm 1rem = 0.423cm rem → mm 1rem = 4.23mm rem → pt 1rem = 12pt rem → pc 1rem = 1pc rem → vw 基于视窗宽度 rem → vh 基于视窗高度 rem → ft 1rem = 0.014ft

EM

相对于父元素的字体大小的单位转换器

em → px 1em = 16px em → rem 1em = 1rem em → inches 1em = 0.167in em → cm 1em = 0.423cm em → mm 1em = 4.23mm em → pt 1em = 12pt em → pc 1em = 1pc em → ft 1em = 0.0139ft em → vw 基于视窗宽度 em → vh 基于视窗高度 em → % 基于父元素

POINT (PT)

点单位转换器

pt → px 1pt = 1.33px pt → em 基于父元素字体大小 pt → rem 基于根元素字体大小 pt → inches 72pt = 1in pt → cm 28.35pt = 1cm pt → mm 2.835pt = 1mm pt → pc 12pt = 1pc pt → ft 864pt = 1ft pt → vw 基于视窗宽度 pt → vh 基于视窗高度 pt → % 基于父元素

CENTIMETER (CM)

厘米单位转换器

cm → pt 1cm = 28.35pt cm → px 1cm = 37.8px cm → em 1cm = 2.36em cm → rem 1cm = 2.36rem cm → inches 2.54cm = 1in cm → mm 1cm = 10mm cm → pc 1cm = 2.36pc cm → ft 30.48cm = 1ft cm → vw 基于视窗宽度 cm → vh 基于视窗高度 cm → % 基于父元素

VIEWPORT WIDTH (VW)

视口宽度单位转换器

vw → cm 1vw = 0.01cm (假设视口宽度为 1000px) vw → px 1vw = 10px (假设视口宽度为 1000px) vw → em 基于父元素的字体大小 vw → rem 基于根元素的字体大小 vw → in 1vw = 0.0328in (假设视口宽度为 1000px) vw → mm 1vw = 0.254mm (假设视口宽度为 1000px) vw → pt 1vw = 0.75pt (假设视口宽度为 1000px) vw → ft 1vw = 0.0328ft (假设视口宽度为 1000px) vw → % 1vw = 1% vw → vh 基于视口高度 vw → pc 1vw = 0.01pc (假设视口宽度为 1000px)

VIEWPORT HEIGHT (VH)

视口高度单位转换器

vh → px 1vh = 10px (假设视口高度为 1000px) vh → em 基于父元素的字体大小 vh → rem 基于根元素的字体大小 vh → in 1vh = 0.0328in (假设视口高度为 1000px) vh → mm 1vh = 0.254mm (假设视口高度为 1000px) vh → pt 1vh = 0.75pt (假设视口高度为 1000px) vh → ft 1vh = 0.0328ft (假设视口高度为 1000px) vh → % 1vh = 1% vh → vw 基于视口宽度

MILLIMETER (MM)

毫米单位转换器

mm → px 1mm = 3.779px (假设96dpi) mm → em 基于父元素的字体大小 mm → rem 基于根元素的字体大小 mm → in 1mm = 0.03937in mm → pt 1mm = 2.8346pt mm → ft 1mm = 0.003281ft mm → % 基于某个基准 mm → vh 基于视口高度 mm → vw 基于视口宽度

PICA (PC)

派卡单位转换器

pc → mm 1pc = 16.6667mm pc → px 1pc = 16px pc → in 1pc = 0.1667in pc → pt 1pc = 12pt pc → em 基于父元素的字体大小 pc → rem 基于根元素的字体大小 pc → ft 1pc = 0.0139ft pc → % 基于某个基准 pc → vh 基于视口高度 pc → vw 基于视口宽度

百分比 (%)

百分比单位转换器

% → pc 100% = 1pc % → vh 50% = 50vh % → vw 50% = 50vw % → em 100% = 1em % → rem 100% = 1rem % → pt 100% = 12pt % → mm 100% = 25.4mm % → in 100% = 1in % → ft 100% = 0.0833ft % → px 100% = 16px (假设基准字体大小为 16px) % → cm 100% = 2.54cm

纸张尺寸

纸张尺寸转换器

A4 → px 210mm x 297mm Letter → px 8.5in x 11in Legal → px 8.5in x 14in A3 → px 297mm x 420mm A5 → px 148mm x 210mm
