CSS单位转换器

支持各种CSS单位之间的相互转换，让前端开发更轻松

PIXEL (PX)

像素单位转换器

px → inches 96px = 1in px → rem 16px = 1rem px → em 16px = 1em px → cm 37.8px = 1cm px → mm 3.78px = 1mm px → pt 1.33px = 1pt px → pc 16px = 1pc px → vw 基于视窗宽度 px → vh 基于视窗高度 px → ft 1152px = 1ft px → % 基于父元素

INCH (IN)

英寸单位转换器

in → pixels 1in = 96px in → cm 1in = 2.54cm in → mm 1in = 25.4mm in → pt 1in = 72pt in → pc 1in = 6pc in → ft 12in = 1ft in → rem 1in = 6rem in → em 1in = 6em in → vw 基于视窗宽度 in → vh 基于视窗高度 in → % 基于父元素
